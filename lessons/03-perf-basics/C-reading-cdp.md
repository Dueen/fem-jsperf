### Lets launch our example project and check out the tools
To launch the project, you need two commands to run

#### The Server
```
tsc && node --inspect dist/server.js --logPath /tmp/foo
```

#### The Client
```
cargo run --release -- -q 1000 -g 100000 -t 2
```

#### Open up chrome
